# 整数安全

*16内存与安全*

参考来源：[SEI CERT INT30-C — Ensure that unsigned integer operations do not wrap](https://wiki.sei.cmu.edu/confluence/display/c/INT30-C)、
    [INT31-C — Ensure that integer conversions do not result in lost or misinterpreted data](https://wiki.sei.cmu.edu/confluence/display/c/INT31-C)

## 16.1 整数溢出检测

**示例：安全整数运算模式**

```c
#include <stdint.h>
#include <limits.h>

/* ✅ 有符号整数加法溢出检测（C99 无符号绕回是定义行为）*/
bool safe_add_i32(int32_t a, int32_t b, int32_t *result)
{
    if ((b > 0 && a > INT32_MAX - b) ||
        (b < 0 && a < INT32_MIN - b)) {
        return false;  /* 溢出 */
    }
    *result = a + b;
    return true;
}

/* ✅ 乘法溢出检测（防止内存分配大小溢出）*/
bool safe_mul_size(size_t a, size_t b, size_t *result)
{
    if (a != 0U && b > SIZE_MAX / a) {
        return false;
    }
    *result = a * b;
    return true;
}

/* ✅ 整数转换：显式检查范围 */
bool safe_cast_i32_to_u16(int32_t val, uint16_t *out)
{
    if (val < 0 || val > UINT16_MAX) {
        return false;
    }
    *out = (uint16_t)val;
    return true;
}
```


## 16.2 位运算安全规则

| 操作 | 规范要求 |
| --- | --- |
| 移位操作 | 移位量必须 ≥ 0 且 < 操作数位宽；对有符号数左移结果溢出为 UB |
| 位操作符 | 使用无符号类型；`~` 对有符号数可能产生意外值 |
| 混合符号运算 | 有符号与无符号混合运算前显式转换（编译器开启 `-Wsign-conversion`） |
