# 权威参考来源

*21参考资料*

参考来源：本文档基于以下权威来源整理，所有规范均可追溯至原文。

## 标准规范

- [ISO/IEC 9899:1999 — Programming Languages — C (C99)](https://www.iso.org/standard/29237.html) — 国际标准，C99 正式规范
- [ISO WG14 — C Standard Working Group](https://www.open-std.org/JTC1/SC22/WG14/) — C 语言标准委员会官网
- [SEI CERT C Coding Standard](https://wiki.sei.cmu.edu/confluence/display/c/SEI+CERT+C+Coding+Standard) — CMU 软件工程研究所，涵盖 89 条规则 + 132 条建议
- [MISRA C:2012](https://www.misra.org.uk/) — 汽车/安全关键系统 C 编码标准（工业界权威）


## 开源项目规范

- [Linux Kernel Coding Style](https://www.kernel.org/doc/Documentation/process/coding-style.rst) — Linus Torvalds，涵盖缩进、命名、宏、注释等全方面
- [GNU Coding Standards (PDF)](https://gnu.net.cn/prep/standards/standards.pdf) — Richard Stallman et al.，最后更新 2024-05-26
- [GNOME C Coding Style](https://developer.gnome.org/documentation/guidelines/programming/coding-style.html) — GTK/GNOME 项目规范


## 技术参考

- [The Lost Art of Structure Packing — Eric S. Raymond](http://www.catb.org/esr/structure-packing/) — 结构体对齐与内存优化权威指南
- [C Struct Padding Initialization — Memfault](https://interrupt.memfault.com/blog/c-struct-padding-initialization) — 结构体填充详解
- [Portable Fixed-Width Integers in C — Barr Group](https://barrgroup.com/embedded-systems/how-to/c-fixed-width-integers-c99) — C99 整数类型嵌入式实践
- [Struct Member Alignment — LinuxVox](https://linuxvox.com/blog/struct-member-alignment-is-it-possible-to-assume-no-padding/) — 跨平台对齐分析
