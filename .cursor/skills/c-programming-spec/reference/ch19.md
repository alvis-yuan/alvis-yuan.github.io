## 19.2 可移植性编码规则

| 规则 | 不可移植写法 | 可移植写法 |
| --- | --- | --- |
| 整数大小 | `int x;`（平台相关） | `int32_t x;`（精确宽度） |
| 字节序 | 直接内存转换 | 使用 `htonl()`/`ntohl()` |
| 函数参数顺序 | `f(i++, i++)` | 分开语句避免未规定求值顺序 |
| 位域符号性 | `int bits:4;` | `uint32_t bits:4;` |
| 字符符号性 | `char c; if(c > 127)` | `unsigned char c;` 或 `uint8_t` |
| 编译器扩展 | `__typeof__`、`__builtin_*` | 用条件编译隔离，提供标准后备实现 |
