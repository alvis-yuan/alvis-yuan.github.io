## 15.1 指针安全规范

**图15.1 — 指针安全生命周期**

- 指针生命周期与安全规则
- 声明 = NULL
- 分配/赋值
- 有效使用
- free()
- → NULL
- ❌ 禁止
- 使用未初始化指针
- （野指针）
- 解引用前不检查 NULL
- （空指针解引用）
- free 后继续使用
- （悬挂指针）
- 指针算术：只在同一数组对象内操作；结果必须在 [base, base+size] 范围内（C99 §6.5.6）
- 越界指针（即使不解引用）的行为在 C99 中是未定义行为

**示例：指针安全综合示例**

```c
/* 所有权明确：谁分配谁释放 */
typedef struct {
    uint8_t *data;       /* owner: 本结构体 */
    size_t   len;
} owned_buf_t;

int32_t owned_buf_create(owned_buf_t *self, size_t len)
{
    assert(self != NULL);
    assert(len > 0U);

    self->data = (uint8_t *)malloc(len);
    if (self->data == NULL) {
        self->len = 0U;
        return NET_ERR_NOMEM;
    }
    self->len = len;
    return NET_OK;
}

void owned_buf_destroy(owned_buf_t *self)
{
    if (self != NULL) {
        SAFE_FREE(self->data);   /* 置 NULL */
        self->len = 0U;
    }
}
```
