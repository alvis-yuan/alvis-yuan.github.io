# 字符串与缓冲区

*17内存与安全*

参考来源：[SEI CERT STR31-C — Guarantee that storage for strings has sufficient space for character data and the null terminator](https://wiki.sei.cmu.edu/confluence/display/c/STR31-C)、
    [STR32-C](https://wiki.sei.cmu.edu/confluence/display/c/STR32-C)

## 17.1 字符串操作安全规范

| 不安全函数 | 安全替代 | 说明 |
| --- | --- | --- |
| `strcpy(dst, src)` | `strncpy(dst, src, n-1); dst[n-1]='\0'` | 防止目标溢出 |
| `strcat(dst, src)` | `strncat(dst, src, n - strlen(dst) - 1)` | 防止拼接溢出 |
| `sprintf(buf, fmt, ...)` | `snprintf(buf, sizeof(buf), fmt, ...)` | 限制输出大小 |
| `gets(buf)` | `fgets(buf, sizeof(buf), stdin)` | `gets` 已在 C11 移除 |
| `scanf("%s", buf)` | `scanf("%127s", buf)` | 限制输入宽度 |

**示例：安全字符串操作示例**

```c
#include <string.h>
#include <stdio.h>

void safe_string_example(const char *input)
{
    char buf[64];
    size_t input_len;

    if (input == NULL) { return; }

    input_len = strlen(input);
    if (input_len >= sizeof(buf)) {
        LOG_ERROR("Input too long: %zu bytes", input_len);
        return;
    }

    /* 安全复制，确保 NUL 结尾 */
    (void)memcpy(buf, input, input_len + 1U);  /* +1 含NUL */

    /* 格式化输出限制大小 */
    char out[128];
    int n = snprintf(out, sizeof(out), "prefix: %s", buf);
    if (n < 0 || (size_t)n >= sizeof(out)) {
        LOG_ERROR("Output truncated");
    }
}
```
