# 中介者模式（Mediator）

用一个中介对象来封装一系列的对象交互。中介者使各对象不需要显式地相互引用，从而使其耦合松散，而且可以独立地改变它们之间的交互。 — GoF《设计模式》

### 📐 原理 & 应用

中介者集中处理所有组件（Colleague）之间的通信，各组件只与中介者交互，不直接引用彼此。

### Redis服务器 — 客户端中介

Redis服务器 — 客户端中介
          
        
        Redis 服务器本身是中介者：所有客户端不直接通信，所有命令通过 Redis 服务器中介转发和协调。Pub/Sub 频道的消息传递也是中介者模式。
