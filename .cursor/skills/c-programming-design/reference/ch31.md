# 迭代器模式（Iterator）

提供一种方法顺序访问一个聚合对象中各个元素，而又不需暴露该对象的内部表示。 — GoF《设计模式》

### 💻 C语言实现

**示例：C — iterator.c（基于 mohitmishra786.github.io）**

```c
typedef struct Iterator {
    bool (*hasNext)(struct Iterator*);
    int  (*next)(struct Iterator*);
} Iterator;

/* Redis dictIterator 遍历哈希表 */
dictIterator* iter = dictGetIterator(dict);
dictEntry* entry;
while ((entry = dictNext(iter)) != NULL) {
    /* 访问 key/value，不需知道内部结构 */
}
dictReleaseIterator(iter);
```
