# 备忘录模式（Memento）

在不破坏封装性的前提下，捕获一个对象的内部状态，并在该对象之外保存这个状态。这样以后就可将该对象恢复到原先保存的状态。 — GoF《设计模式》

### 🔍 C开源项目应用

### Redis — RDB 快照持久化

Redis — RDB 快照持久化
          
        
        Redis 的 RDB（Redis Database）持久化是备忘录模式：定期捕获内存数据库的完整状态（备忘录），存储为 .rdb 文件。崩溃恢复时，将对象恢复到快照时刻的状态。
        **示例：Redis RDB — 状态快照（备忘录）**

```c
/* 创建备忘录：捕获当前内存状态 */
rdbSaveBackground(server.rdb_filename, &rsi);
  → fork() + rdbSave() /* 子进程快照写磁盘 */

/* 恢复备忘录：从快照恢复状态 */
rdbLoad(server.rdb_filename, &rsi, RDBFLAGS_NONE);
```
