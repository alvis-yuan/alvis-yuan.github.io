部署到 GitHub Pages
  • 将此仓库推到 GitHub（如已在 GitHub 上，则跳过）。
  • 在本地运行：git add index.htm && git commit -m "Add index.htm" && git push
  • 在 GitHub 仓库页面：Settings → Pages，选择分支（例如 main）和根目录（/），保存后几分钟将可访问。
