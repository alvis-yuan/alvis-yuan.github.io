Core 6.0 — Attribute Protocol (ATT)

本索引覆盖 GATT 下层 Attribute Protocol 全部标准 PDU，与 Attribute Protocol Specification 中 Table 3.43（Opcode 汇总）、§3.3（PDU 格式）及 §3.4（分过程定义）对齐，共 31 条 ATT PDU。

离线：单文件自包含；规范链接需联网。

PDU 格式 (Table 3.2)

Attribute Opcode (1) + Parameters (0…ATT_MTU−X) [+ Auth Signature 12]。Opcode bit7=签名，bit6=命令。

PDU 类型 (Table 3.1)

REQ/RSP、CMD（无响应）、NTF、IND/CFM。服务器至少支持 FIND_INFORMATION_REQ 与 READ_REQ。

请求-响应 (Table 3.44)

各 REQ 的成功响应见卡片「配对响应」；失败统一 ATT_ERROR_RSP（§3.4.1）。
