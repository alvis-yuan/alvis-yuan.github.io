§1 范围 (Scope)

本节定义了蓝牙低功耗（Bluetooth Low Energy, BLE）物理层（PHY）的整体范围与基本架构。BLE 设备工作在 2.4 GHz ISM 免许可频段（工业、科学和医疗），使用跳频收发机（Frequency Hopping Transceiver）以抵抗干扰与衰落。

调制方案概览

规范定义了两种调制方案，同时引入了用于信道探测（Channel Sounding, CS）的幅移键控（ASK）：

💡 核心要点： LE 设备必须支持 LE 1M PHY；LE Coded、LE 2M 和 LE 2M 2BT 均为可选。所有 PHY 均采用时分双工（TDD）通信方式。支持 Channel Sounding 的设备还需支持 ASK 调制用于测距。

§2 频段与信道安排 (Frequency Bands and Channel Arrangement)

BLE 系统工作在 2400 MHz ~ 2483.5 MHz 的 ISM 频段，共定义了 40 个 RF 信道，中心频率间隔 2 MHz，从 2402 MHz 到 2480 MHz。

支持 Channel Sounding（CS）的设备还额外使用 72 个 RF 信道（中心频率 = 2402 + k MHz，k 为 2~22 及 26~76 的整数）。

信道频率计算

f_channel(n) = 2402 + 2n [MHz], n = 0, 1, …, 39

§3 发射机特性 (Transmitter Characteristics)

本节规定了 BLE 发射机在天线连接器处（或等效辐射点）的各项技术要求，涵盖功率、调制、杂散发射和频率容差等关键指标。

发射功率范围

P_max ∈ [0.01 mW (−20 dBm), 100 mW (+20 dBm)]

功率等级划分

根据最大辐射发射功率（Pmax）将设备分为四个功率等级：

Class 1
+10 dBm < Pmax ≤ +20 dBm (最高 100 mW)
Class 1.5
+4 dBm < Pmax ≤ +10 dBm (最高 10 mW)
Class 2
0 dBm < Pmax ≤ +4 dBm (最高 2.5 mW)
Class 3
−20 dBm ≤ Pmax ≤ 0 dBm (最高 1 mW)
⚡ 功率控制： 规范引入了 LE Power Control Request 特性，允许已连接设备根据接收信号强度（RSSI）动态调节对端的发射功率，以避免接收机饱和、降低功耗并减少干扰。相邻发射功率级之间的间隔建议不超过 8 dB。

§3.1 调制特性 (Modulation Characteristics)

BLE 采用 高斯频移键控（GFSK） 调制方式，带宽-比特周期积（BT）= 0.5，调制指数 h 在 0.45 ~ 0.55 之间。

  • 二进制 1 → 正频率偏移（载波升高）
  • 二进制 0 → 负频率偏移（载波降低）

§3.1.1 稳定调制指数 (Stable Modulation Index)

BLE 区分了两类调制指数规格：

类型调制指数范围特点
标准调制指数0.45 ~ 0.55默认要求，容差较宽
稳定调制指数0.495 ~ 0.505更精确，接收机可优化解调
📌 作用： 当发射机具有稳定调制指数时，可通过特性支持机制告知接收方，使接收方调整解调算法，从而在弱信号环境下获得更好的接收性能。该特性必须适用于设备支持的所有 LE 发射机 PHY。

§3.1.2 信道探测步骤的调制特性 (CS Modulation)

当 CS_SYNC 包使用 LE 2M 2BT PHY 传输时，调制规则与标准 GFSK 基本相同，但有以下重要差异：

  • 带宽-比特周期积 BT = 2.0（而非 0.5），意味着高斯滤波器更宽，脉冲间干扰更小
  • 符号中心处的最小频偏 ≥ 420 kHz（而非 185 kHz）
  • 2 Msym/s 符号率
🎯 为何 BT=2.0？ Channel Sounding 的核心目的是精确测距，需要载波相位在整个测量周期内保持高度稳定。更大的 BT 值意味着高斯滤波器更宽（时域更短），能减少符号间干扰，使相位变化更平缓，从而大幅提升相位测量精度。

§3.1.3 CS 步骤的 SNR 控制 (SNR Control)

用于 Channel Sounding 的设备可支持对调制传输的 信噪比（SNR）输出进行精确控制，共定义 5 个 SNR 输出级别（SOI 0~4）。

🔬 SNR 控制的用途： 在 CS 测距场景中，精确控制发射信噪比能使接收方在已知 SNR 条件下评估测量质量，有助于提高距离估计算法的鲁棒性，也可用于模拟不同的信道环境进行测试。

§3.2 杂散发射 (Spurious Emissions)

杂散发射要求确保 BLE 发射机不会对其他信道和频段造成过度干扰，分为调制频谱、带内杂散和带外杂散三个层面。

§3.2.1 调制频谱 (Modulation Spectrum)

遵从 FCC Part 15.247 规定时，发射机调制信号的 6 dB 最小带宽须在 100 kHz 分辨率带宽下测量，且不得小于 500 kHz。

§3.2.2 带内杂散发射 (In-band Spurious Emission)

规范对距载波一定频率偏移处的相邻信道功率（以 1 MHz 带宽内总功率计）做出严格限制：

§3.2.3 带外杂散发射 (Out-of-band Spurious Emission)

2.4 GHz ISM 频段以外的杂散发射要求由设备制造商依据目标销售国的法规自行负责，蓝牙规范不作统一规定（各国 EMC/无线电法规可能不同）。

⚠️ 注意： 设备制造商需参考目标市场的无线电法规（如欧盟 CE RED、美国 FCC、中国 SRRC 等），并确保带外杂散发射满足当地要求。

§3.3 射频频率容差 (Radio Frequency Tolerance)

规范对发射机的频率精度和漂移有严格限制，以确保接收机能稳定解调。

参数限制值
中心频率偏差（含初始偏移 + 漂移）±150 kHz
包内最大频率漂移±50 kHz
最大漂移速率400 Hz/μs
📐 为何需要频率容差？ BLE 信道间隔为 2 MHz，过大的频率偏移会导致信号落入相邻信道，引起干扰或解调失败。±150 kHz 的总容差包含了设备的初始频率偏移（晶振精度）和工作过程中的温度漂移。

§3.4 稳定相位 (Stable Phase)

支持 Channel Sounding 的设备须能在 T_PM_MEAS 时间段内保持 RF 信号相位的高度稳定。

相位稳定性通过以下步骤评估：对采样相位进行去线性趋势（去除频率偏移导致的线性相位增长），然后计算残余相位的零均值去趋势序列。

φ_zmd[n] = φ_d[n] − φ̄_d
📏 要求： 95% 的 |φ_zmd[n]| 绝对值须 ≤ 20 度。这保证了 CS 测距的相位测量具有足够的一致性。

§3.5 CS 频率测量与生成 (Frequency Measurement and Generation in CS)

Channel Sounding 子事件由多个 CS 步骤组成，每个步骤使用特定的 CS 信道频率。本节定义了频率生成和测量的参数框架。

  • 每个 CS 子事件包含 M 个 mode-0 步骤 + K 个非 mode-0 步骤
  • 每个包含 CS 音调的步骤有 P 个相位测量周期（1 ≤ P ≤ N_AP + 扩展）
  • N_AP 为天线路径数，最多 4 个

§3.5.1 分数频率偏移 (Fractional Frequency Offset)

分数频率偏移（FFO）描述实际发射频率与标称频率之间的误差，直接影响相位测量的准确性。FFO 须在规范规定的容差范围内，以保证测距精度。

🔗 关联： FFO 与 §6.3 的频率激励误差校正（LFAE 校正）密切相关。设备需在相位计算中补偿已知的频率误差，以提高 Phase-based Ranging 的精度。

§3.5.2 期望发射频率 (Expected Transmitted Frequencies)

规范明确定义了每个 CS 步骤的期望发射频率，用于接收方的参考解调（见 §6.1 参考接收机定义）。实际发射频率与期望频率的偏差须在 §3.3 所规定的容差范围内。

§4 接收机特性 (Receiver Characteristics)

本节规定了 BLE 接收机的核心性能指标，涵盖灵敏度、抗干扰性能和 CS 接收机特性，是保障通信质量和链路可靠性的关键。

§4.1 接收灵敏度 (Sensitivity)

接收灵敏度是指接收机能够成功解调信号（误包率 BER ≤ 0.1%）时所需的最低输入功率。

PHY 类型最低接收灵敏度数据速率
LE 1M−70 dBm1 Mb/s
LE 2M−70 dBm2 Mb/s
LE Coded (S=8)−82 dBm125 kb/s
LE Coded (S=2)−75 dBm500 kb/s
📡 LE Coded PHY 的优势： 通过前向纠错（FEC）编码，LE Coded PHY 将有效通信距离提升了约 4 倍（S=8 相比 LE 1M 灵敏度提升约 12 dB），非常适合工业物联网、资产追踪等长距离低速率应用场景。

§4.2 干扰性能 (Interference Performance)

接收机须在存在同频干扰、相邻信道干扰和带外干扰的情况下保持可接受的误包率。规范通过共存比（C/I）和阻塞（Blocking）测试来验证。

§4.3 CS 接收机特性 (CS Receiver Characteristics)

Channel Sounding 接收机除满足标准 BLE 接收要求外，还需支持：

  • 在 −70 dBm 输入电平下完成精确的相位测量
  • 对 IQ 采样进行处理以支持相位差计算
  • 结合天线切换完成 AoA/AoD 或 PBRT（基于相位的测距）

§5 天线与方向性 (Antenna and Direction Finding)

本节规定了支持到达角（AoA）/ 离开角（AoD）方向定位以及 Channel Sounding 天线切换的要求，是 BLE 方向定位（Direction Finding）和测距功能的基础。

§5.1 AoA/AoD 发射机天线切换 (Transmitter Antenna Switching)

使用恒定音调扩展（Constant Tone Extension, CTE）结构进行 AoA/AoD 测量时，发射机须按照规定的天线切换模式依次使用不同天线发射 CTE 信号。

  • 切换模式由主机（Host）指定，控制器按顺序执行
  • 天线只能在保护期（guard period）和切换槽（switch slots）期间切换
  • 若模式在最后一个采样槽前耗尽，则从头循环重复

§5.2 AoA/AoD 接收机特性 (Receiver Characteristics for AoA/AoD)

AoA/AoD 接收机须在特定测试切换模式下满足相位测量的精度要求。

§5.2.1 基本定义

给定天线 Am 的采样槽 n 的 IQ 样本 I(n) 和 Q(n)，相位定义为：

φ(n) = Arg(I(n) + j·Q(n))

相对相位（n > 2）：

θ(n) = principal(φ(n) − 2φ(n−1) + φ(n−2))

§5.2.2 精度要求

对于每个天线 Am（除参考天线 A0），95% 的相对相位值 v 须满足：

−0.52 ≤ principal(v − MRP(m)) ≤ 0.52

§6 相位测量 (Phase Measurements)

本节定义了 Channel Sounding 中基于相位的测距（Phase-Based Ranging, PBRT）的参考框架和精度要求，是实现厘米级定位精度的核心。

🎯 PBRT 原理： 通过在多个不同频率的 CS 信道上测量发射与接收信号的相位差，利用相位差随频率线性变化的特性（斜率 = 2π × 往返时间），反算出两设备之间的距离。
距离 d = (c × ToF) / 2，ToF = α[p] / (2π)，c = 3×10⁸ m/s

§6.1 参考接收机 (Reference Receiver for Phase-based Ranging)

参考下变频定义为使用期望发射频率 f_E[k] 进行的本振混频，经低通滤波后得到基带复数信号：

x̂(k,p,t) = LPF{ x(p,t) · e^(−j2π·f_E[k]·t) }

其中 x(p,t) 为天线处的射频信号，LPF 消除高频分量。

§6.2 相位测量精度 (Phase Measurement Accuracy)

在满足以下条件时，规范对线性回归解的精度提出要求：

  • CS 音调在所有 CS 信道上至少交换一次
  • 发射信号满足 §3.5.2 的频率偏移条件
  • 接收机输入电平为 −70 dBm

对频率-相位数据点 (f_E[k], θ_C,UW[k,p]) 做线性回归，得到斜率 α[p] 和截距 β[p]，须满足：

|α[p]| < 2π × 1.7 ns （95% 子事件）
|principal(α[p]·f_E[k] + β[p] − θ_C,UW[k,p])| < 0.28 rad （95% 值）

§6.3 频率激励误差校正 (Frequency Actuation Error Correction)

若设备存在本地频率激励误差（LFAE），则必须对其引起的相位旋转进行补偿。LFAE 是指实际生成频率与期望频率之间的偏差。

CS 音调相位校正

∠PCT' = ∠PCT − 2π · Δf_k × T_PM_CENTER_DELTA

CS_SYNC 探测序列相位校正

∠SS_PCT' = ∠SS_PCT − 2π · Δf_k × T_SY_CENTER_DELTA
🔧 校正原理： 频率误差 Δf_k 会在时间 T 内造成 2π·Δf_k·T 的相位积累。通过精确计算发送与接收 CS 音调中心点之间的时间差（T_PM_CENTER_DELTA），可以计算出需要补偿的相位量，从而提高测距精度。

§6.4 相位测量时序 (Phase Measurement Timing)

相位校正项（PCT）在音调的相位测量周期 T_PM 内测量。规范定义了测量区域（Valid Region）以排除天线切换过渡期间的不稳定样本。

  • VSG 和 VSA 与 CS 程序的时序节拍同步
📋 测试流程： VSG 向 IUT 发送 CS 音调；IUT 向 VSA 发送响应的 CS 音调。VSA 捕获的 x(t) 信号包含整个 CS 步骤的完整 RF 波形，用于验证 IUT 的发射频率、相位稳定性和测量精度是否符合规范要求。
