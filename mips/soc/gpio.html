第一性原理：为何需要 GPIO

SoC 引脚数量有限，却需同时服务 CPU 启动、外设通信、中断检测与用户 I/O。GPIO 控制器在同一物理 PAD 上复用多种功能：普通输入/输出、外设复用（alternate function）、以及可独立屏蔽的中断源。手册 29.1 指出 GPIO 用于产生和捕获应用特定的输入/输出信号。

图 1 — GPIO 组汇聚 PAD 配置，分别向 XBurst2 与 RISC-V 提供独立中断 pending/mask

必要 最小引脚控制集合

  • PxPINL — 只读，反映引脚电平
  • PxPAT0/PxPAT1 + PxMSK — 方向与 GPIO/外设功能选择
  • PxINT + PxFLG + PxPEND — 中断使能、源标志与 pending

增强特性 手册列出的扩展能力

  • Shadow / 单 pin / 多 pin 四种配置模式
  • CPU 与 RISC-V 独立 MMSK/PMPEND
  • Drive Strength、Schmitt、Slew Rate、Bus Hold（PE）
  • GPIOA~E 位于 AHB_MCU，多主可并行访问不同组

29.1 概述（Overview）

General Purpose I/O Ports (GPIO) is used in generating and capturing application-specific input and output signals. Each port can be programmed as an output, an input or function port that serves certain peripheral. As input, pull up/down can be enabled/disabled for the port and the port also can be configured as level or edge tripped interrupt source.

29.2 特性（Features）

  •  Each port can be configured as an input, an output or an alternate function port.
  •  Each port can be configured as an interrupt source triggered by high/low level or rising/falling edge or dual edge.
  •  Each group contains two interrupts, which are provided to the CPU and RISC-V respectively. And each interrupt has independent pending and mask register. which can independently control the interrupt response of each port.
  •  There are four ways to configure GPIO function: a) normal configuration mode.
  • b) shadow configuration mode.
  • c) single pin configuration mode.
  • d) multiple pin configuration mode.
  •  Support load shadow to set and clear different bits of register at the same time.
  •  Each group contains two sets of shadow registers, which are provided for CPU and RISC-V respectively.
  •  Each group contains two sets of multiple pin configuration function registers, which are provided for CPU and RISC-V respectively.
  •  GPIOA~E are independent modules on the AHB_MCU bus, And multiple masters can access different groups in parallel.
  •  Each port has an internal pull-up or pull-down resistor connected. The pull-up/down resistor can be disabled.
  •  Each port supports the configuration of Drive Strength、Schmitt and Slew Rate.
  •  The port of GPIOE group supports the configuration of Bus Hold.

29.4 寄存器描述 — 约定（Conventions）

Register Address = Base + Address offset。读/写属性：R（只读）、W（只写）、RW（读写）、RCW、RSW、RWC、RWS、RC、RS、SPEC。复位值：1/0/?（未知）。灰色位域表示该 GPIO 组未使用，可忽略。

29.5.3 四种配置方式

  • normal — 串行写寄存器，存在三态风险
  • shadow — 写 shadow 寄存器后通过 PxGLD0/1 一次性加载
  • single pin — 通过 PxSPCFG 配置单 pin
  • multiple pin — 通过 PxMPSEL + PxMPCFG 配置多 pin

29.5.7 电源域切换

PA00~PA11 和 PD00~PD05 支持 1.8V/3.3V 电源域切换，详见 CPM 章节 EXCLK_DS (0xE0)。
