第一性原理：为何需要外部存储控制器

处理器需要访问片外 NOR Flash、SRAM 等静态存储器。EMC（External Memory Controller）将片外存储空间划分，并输出符合各类静态存储器与总线接口规范的控制信号，使 NOR Flash 等静态存储器可连接至本处理器（见 6.1 Overview）。

图 1 — EMC 将系统总线访问译码为两路片选静态存储 Bank（CS1#、CS2#）

为实现片外静态存储访问，哪些寄存器/特性是「必要」的？

必要 地址映射 + 基本时序

  • SACR1 / SACR2 — 定义静态 Bank 的 BASE 与 MASK，满足 (physical_address[31:24] & MASKi) == BASEi 时 Bank i 激活。
  • SMC0R1 / SMC0R2 — 读/写地址建立、数据捕获、CS# 保持等核心时序（TRAS、TWAS、TCH、TAW_OE、TBP_DS）。
  • SMC1R1 / SMC1R2 — 读/写恢复时间（TRDRV、TWRRV）及 Bank 切换空闲周期（STRV），防止连续访问数据总线冲突。

复位后默认 CS1# 映射 0x1B000000–0x1B00FFFF、CS2# 映射 0x1A000000–0x1A00FFFF（各 64KB）。

增强特性 手册列出的扩展能力

  • SMT 字段 — 支持 Normal Memory、Burst ROM、地址/数据复用（ADM）三种静态存储类型。
  • WAIT# 引脚 — WP_EN 使能时可通过外部 WAIT# 动态延长访问时间。
  • DOUBLE — 将所有等待时间加倍。
  • BL 突发长度 — Burst ROM 模式下 4/8/16/32 次连续访问。
  • 8/16 位总线宽度 — BW 字段可配置每 Bank 数据宽度。

6.1 概述（Overview）

外部存储控制器（EMC）将片外存储空间划分，并输出符合各类静态存储器与总线接口规范的控制信号，使 NOR Flash 等静态存储器可连接至本处理器。

6.2 特性（Features）

  • 每个 Bank 可直接接口 8 位或 16 位（无字节控制）总线宽度的外部存储器接口设备或外部静态存储器。
  • 支持 2 路外部片选 CS2#~CS1#，每个 Bank 可单独配置。
  • 支持地址与数据复用的 NOR Flash 设备。
  • 可灵活定义控制信号的断言与撤销时间。
  • 通过 WAIT 引脚插入等待周期。
  • 自动插入等待周期，防止连续访问不同 Bank 或同一 Bank 读后跟写时的数据总线冲突。
  • 支持的存储器类型：
    • 外部异步存储器（非突发设备）
    • 外部异步突发 ROM
    • 外部异步地址/数据复用存储器

6.6 时序设置（Timing Setting）

6.6.1 普通存储器（Normal Memory）

当 SMCRn（n = 1 至 2）中 SMT 字段为 0 时，Bank n 连接普通存储器（非突发 ROM、Flash、普通 SRAM 或类存储器设备）。访问 Bank n 时，地址输出后 CSn# 即被断言；RD#（可用作 OE#）与写控制信号 WE# 亦被断言。

图 4 — 普通存储器读时序示意（Tras、T1、Toe、Tch、Trdrv）

读时序字段（SMC0R_i）：

  • Tras（SMC0R_i[15:12]）— 地址建立时间：地址信号在 RD# 断言前须保持的最小时间。
  • T1 — 一次访问固有周期。
  • Toe（SMC0R_i[23:20]）— 相对 RD# 断言的数据捕获时间。
  • Tch（SMC0R_i[27:24]）— 相对数据捕获时间的 CS# 撤销时间。
  • Trdrv（SMC1R_i[23:20]）— Bank 切换或同 Bank 读后跟写时插入的空闲周期数（0~63）。

示例：Tras = 3, Toe = 2, Tch = 2, Trdrv = 1，忽略 wait_pin。

写时序字段（Figure 6-6）：

  • Twas（SMC0R_i[11:8]）— 地址建立时间：地址在 WE# 断言前须保持的最小时间。
  • Tds（SMC0R_i[19:16]）— 数据建立时间。
  • T2 — 一次写固有周期。
  • Tch — WE# 最早上升沿起数据信号须保持的最小时间。
  • Twrrv（SMC1R_i[19:16]）— 写恢复时间。

示例：Twas = 3, Tds = 2, Tch = 1, Twrrv = 1，忽略 wait_pin。

WAIT# 监控（Figure 6-8、6-9、6-10）：启用 wait_pin 监控（SMC1R_i[31] 中 WAIT_ENABLE）时，数据捕获前 wait pin 须撤销。Taw（SMC0R_i[23:20]）定义相对 RD# 断言的 WAIT# 捕获时间；Tbp（SMC0R_i[19:16]）定义相对 WE# 断言的 WAIT# 捕获时间。

6.6.2 突发 ROM（Burst ROM）

SMCRn 中 SMT 设为 1 时，Bank n 可连接突发 ROM。突发 ROM 接口为具有 nibble 访问功能的 ROM 提供高速访问。基本访问方式与普通存储器相同，但首周期结束后仅改变地址即执行下一次访问。

  • 8 位突发 ROM：BL1~0 可设连续访问次数为 4、8、16 或 32。
  • 16 位 ROM：同样可设 4、8 或 16。

Figure 6-11：Taw 设置读选通到首数据的延迟；Tbp 设置连续地址到数据的延迟。示例 Taw = 2, Tbp = 1。

Figure 6-12：突发 ROM 写时序与普通存储器相同，但用 Taw 代替 Tbp 设置 WE# 延迟。示例 Taw = 2。

6.6.3 地址/数据复用存储器（Address/Data Multiplexed Memory）

SMCRn 中 SMT 为 2 时，Bank n 连接地址/数据复用存储器，地址与数据可复用在同一外部总线上。

Figure 6-13 读时序示例：Tavds = 1, Tavdp = 2, Tavdh = 1, Tras = 6, Toe = 2, Tch = 2, Trdrv = 1。

Figure 6-14 写时序示例：Tavds = 1, Tavdp = 2, Tavdh = 1, Twas = 6, Tds = 2, Tch = 1, Twrrv = 1。
