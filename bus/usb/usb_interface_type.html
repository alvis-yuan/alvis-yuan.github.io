USB Type-A

USB 2.0

最广泛使用的 USB 接口，扁平矩形设计，仅能单向插入。通常作为主机端（Host）接口，见于电脑、充电头、Hub 等设备。

引脚数 4
速率 480 Mbps
电压 5V
电流 0.5A

USB Type-A 连接器实物图

USB 2.0 官方规范 (usb.org)

引脚定义详解（4 Pin）

引脚 #信号名称类型说明导线颜色
1 VBUS (+5V) 电源 电源正极，提供 +5V 电压，为连接的设备供电，标准电流 500mA（USB 2.0） 红色
2 D− 数据 USB 2.0 差分数据信号负极。与 D+ 构成差分对，用于半双工数据传输（最高 480 Mbps） 白色
3 D+ 数据 USB 2.0 差分数据信号正极。与 D− 共同承载双向数据，通过电压差区分逻辑 0/1 绿色
4 GND 地线 电源与信号公共地，为整个电路提供参考电位，保证信号完整性 黑色
外壳 Shield 屏蔽 金属外壳屏蔽层，接大地（Chassis GND），减少电磁干扰（EMI） —

💡 应用场景

键盘、鼠标、U盘、USB Hub、充电头插口、早期外部存储设备。Type-A 接口仅允许一个方向插入，误插力量过大会损坏接口。

USB Type-B

USB 2.0

近似正方形接口，顶部有两个斜切角。主要用于设备端（Device），常见于打印机、扫描仪、音频接口等大型外设，机械强度高，耐插拔。

引脚数 4
速率 480 Mbps
电压 5V
电流 0.5A

USB Type-B 接口常见于打印机等大型外设

USB 2.0 官方规范 (usb.org)

引脚定义详解（4 Pin）

引脚 #信号名称类型说明导线颜色
1 VBUS (+5V) 电源 电源正极 +5V，为设备供电，Type-B 设备通常从主机端获取总线供电 红色
2 D− 数据 差分数据信号负极，与 D+ 共同承载 USB 2.0 数据（最高 480 Mbps 高速模式） 白色
3 D+ 数据 差分数据信号正极，通过检测 D+/D− 上的上拉电阻判断设备速度类型 绿色
4 GND 地线 公共地线，信号和电源返回路径 黑色

💡 应用场景

打印机、扫描仪、MIDI 音频接口、早期外置硬盘盒。Type-B 接口机械结构稳固，适合需要频繁插拔的工业/办公环境，耐用性优于 Mini/Micro USB。

Mini-USB

USB 2.0

Mini-USB（Mini-B 最常见）是早期便携设备的标准接口，比 Type-A/B 更小，有 5 个引脚（增加了 ID 引脚，支持 OTG）。现已被 Micro-USB 取代。

引脚数 5
速率 480 Mbps
电压 5V
OTG 支持

Mini-USB 接口曾广泛用于早期数码相机

USB 2.0 Mini 规范 (usb.org)

引脚定义详解（5 Pin）

引脚 #信号名称类型说明导线颜色
1 VBUS (+5V) 电源 电源正极 +5V，为连接设备供电，最大电流 500mA 红色
2 D− 数据 差分数据负极，与 D+ 组成传输对，实现 USB 2.0 高速数据通信 白色
3 D+ 数据 差分数据正极，通过 D+/D− 差分电压传输数据信号 绿色
4 ID 控制 OTG 身份识别引脚：Mini-A 端接 GND（作为主机），Mini-B 端悬空（作为设备）。用于 USB On-The-Go 主从切换 —
5 GND 地线 公共地，信号与电源的返回路径 黑色

⚠️ 关于 ID 引脚（OTG）

ID 引脚是 Mini/Micro USB 相对标准 Type-A/B 新增的引脚，用于实现 USB On-The-Go（OTG）功能。当 ID 接地时，设备作为 Host；悬空时作为 Device。这使得如手机、相机可直接读取 U 盘等外设，无需电脑中转。

💡 应用场景

早期数码相机（佳能、索尼）、MP3 播放器、GPS 导航仪、部分早期功能手机、PSP 游戏机。已基本被 Micro-USB 替代。

Micro-USB

USB 2.0 USB 3.0

Micro-USB 是曾经最普及的移动设备接口，比 Mini-USB 更薄更耐用（标称 10,000 次插拔）。Micro-B 2.0 有 5 针，而 Micro-B 3.0 则扩展至 10 针。

Micro-B USB 2.0（5 Pin）

2.0引脚 5
3.0引脚 10
速率 5 Gbps
OTG 支持

Micro-USB 曾是安卓手机标配接口

📌 USB 3.0 Micro-B 设计要点

Micro-B 3.0 接口在 Micro-B 2.0 接口旁边增加了一个额外的 5 针扩展区域（引脚 6-10），从而实现后向兼容。外形比 Micro-B 2.0 宽约一倍，可轻松辨别。常见于早期三星 Galaxy 手机和 USB 3.0 移动硬盘。

USB Type-C

USB 3.x USB4

可翻转对称设计，24 个引脚，支持 USB4（40Gbps）、240W 供电（EPR）、DisplayPort/HDMI 视频输出、Thunderbolt 4 等多种协议复用。是当前最主流的现代接口。

引脚数 24
速率 40 Gbps
功率 240 W
可翻转 ✓

USB Type-C 椭圆形可翻转接口

USB Type-C 规范 Rev 2.1 (usb.org) USB Power Delivery 规范 (usb.org)

Side A 引脚（A1 ~ A12）

引脚信号名称类型说明
A1GND地线公共地，电源和信号返回路径
A2TX1+超速SuperSpeed 差分发送正极（USB 3.x/USB4 高速数据发送 TX）
A3TX1−超速SuperSpeed 差分发送负极，与 TX1+ 构成发送差分对
A4VBUS电源电源总线，标准 5V 或 PD 协商后可达 5V-20V（EPR 可达 48V）
A5CC1控制配置通道 1（Configuration Channel）：用于接口插入检测、主从协商、Power Delivery 通信及电缆方向识别
A6D+数据USB 2.0 差分数据正极（向后兼容 USB 2.0 及以下协议）
A7D−数据USB 2.0 差分数据负极
A8SBU1旁路Sideband Use 1：旁路信号，用于 Alternate Mode（如 DisplayPort AUX、音频 GND）
A9VBUS电源第二路 VBUS，与 A4 并联增大载流能力（多引脚分担电流）
A10RX2−超速SuperSpeed 差分接收负极（接收来自对端的高速数据）
A11RX2+超速SuperSpeed 差分接收正极，与 RX2− 构成接收差分对
A12GND地线公共地

Side B 引脚（B1 ~ B12，镜像对称）

引脚信号名称类型说明
B1GND地线公共地（与 A1 对称，保证翻转后信号完整）
B2TX2+超速第二组 SuperSpeed 发送正极，当线缆翻转时启用（A 侧 TX1 切换为 B 侧 TX2）
B3TX2−超速第二组 SuperSpeed 发送负极
B4VBUS电源电源总线（与 A4 并联）
B5CC2控制配置通道 2：当线缆翻转时，CC2 替代 CC1 承担 PD 通信；未翻转时用于电缆 VCONN 供电检测
B6D+数据USB 2.0 数据正极（与 A6 在接口内部连接）
B7D−数据USB 2.0 数据负极（与 A7 在接口内部连接）
B8SBU2旁路Sideband Use 2：翻转后的旁路信号通道，与 SBU1 互为镜像
B9VBUS电源第四路 VBUS，多路并联保证大电流输出（240W EPR 模式下总电流 5A）
B10RX1−超速第一组 SuperSpeed 接收负极（与 A 侧 RX2 对应，翻转后切换使用）
B11RX1+超速第一组 SuperSpeed 接收正极
B12GND地线公共地

🔄 可翻转设计原理

Type-C 两侧引脚镜像对称。CC1/CC2 用于检测插入方向：当识别到哪一侧被插入，控制器即激活对应一侧的 TX/RX 差分对，同时将另一侧的 CC 用于 VCONN 为有源线缆供电，从而实现无论正反插均可正常工作。

USB 3.x Type-A

SuperSpeed

外形与 USB 2.0 Type-A 相同，内部增加 5 个 SuperSpeed 引脚，蓝色舌片是识别特征。向下兼容 USB 2.0 设备，广泛用于现代 PC、笔记本和 U 盘。

引脚数 9
速率 5~20 Gbps
电压 5V
电流 0.9A

蓝色内芯是 USB 3.x Type-A 的识别标志

USB 3.2 官方规范 (usb.org)

引脚定义详解（9 Pin）

引脚 #信号名称类型USB 版本说明
1 VBUS (+5V) 电源 USB 2.0+ 电源正极 +5V，USB 2.0 最大 500mA，USB 3.0 最大 900mA
2 D− 数据 USB 2.0+ USB 2.0 差分数据负极，用于后向兼容，保证与旧设备通信
3 D+ 数据 USB 2.0+ USB 2.0 差分数据正极
4 GND 地线 USB 2.0+ 公共地，电源与 USB 2.0 信号的参考地
5 StdA_SSRX− 超速 USB 3.0+ SuperSpeed 接收差分负极，接收来自设备的高速数据（5~20 Gbps）
6 StdA_SSRX+ 超速 USB 3.0+ SuperSpeed 接收差分正极，与 Pin5 构成全双工接收对
7 GND_DRAIN 地线 USB 3.0+ SuperSpeed 信号专用接地，减少高频信号的串扰和阻抗不匹配，保证信号完整性
8 StdA_SSTX− 超速 USB 3.0+ SuperSpeed 发送差分负极，向设备发送高速数据
9 StdA_SSTX+ 超速 USB 3.0+ SuperSpeed 发送差分正极，与 Pin8 构成全双工发送对（独立于接收对，支持同时收发）

🔄 后向兼容性

USB 3.x Type-A 完全兼容 USB 2.0 设备。当接入 USB 2.0 设备时，仅使用 Pin1-4；接入 USB 3.0 设备时，Pin5-9 同时激活实现 SuperSpeed 传输。

💡 应用场景

USB 3.x U 盘、外置 SSD、USB 3.0 集线器（Hub）、网卡、摄像头、现代 PC/Mac 的 USB-A 口。速率支持 USB 3.0（5G）/ 3.1（10G）/ 3.2（20G）。

USB 3.x Type-B

SuperSpeed

USB 3.0 Type-B 是标准 Type-B 的升级版，在正方形外壳顶部增加了一个额外的凸出区域（容纳 5 个 SuperSpeed 引脚），共 9 个引脚。外形更高，不向下兼容 2.0 Type-B 线缆插入 3.0 Type-B 口。

引脚数 9
速率 5 Gbps
电压 5V
兼容 2.0 设备

USB 3.x Type-B 常见于高速外置硬盘及打印机

⚠️ 兼容性注意

USB 3.0 Type-B 插头无法插入 USB 2.0 Type-B 插座（外形不兼容）；但 USB 2.0 Type-B 插头可以插入 USB 3.0 Type-B 插座（降速至 USB 2.0 运行）。

💡 应用场景

USB 3.0 外置硬盘、高速扫描仪、工业相机、台式打印机（配备 USB 3.0 的机型）、USB 3.0 测试仪器。
