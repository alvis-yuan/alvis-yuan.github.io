📡引言与基本概念

调制（Modulation）是将信息信号加载到高频载波上以便传输的过程；解调（Demodulation）则是在接收端恢复原始信息的逆过程。调制是无线通信系统的物理层核心，决定了系统的频谱效率、功率效率、抗干扰能力和实现复杂度。

📌 为什么需要调制？
基带信号频率低，天线辐射效率与频率成正比。直接发射低频信号需要极长天线（如30Hz信号需要5000km天线），调制将信息"搭载"到高频载波上，实现高效辐射、多路复用和频段分配。

📻模拟调制：AM / FM / PM

模拟调制是无线通信的起点，信息信号以连续方式改变载波的幅度、频率或相位。1G移动网络（NMT、AMPS）即采用频率调制（FM）传输模拟语音。

AM

调幅（Amplitude Modulation）

用信息信号调制载波幅度。最早商用广播技术（1920年代），中波/短波广播标准。

FM

调频（Frequency Modulation）

用信息信号调制载波频率。1G蜂窝（AMPS/NMT）核心技术，抗幅度噪声强。

PM

调相（Phase Modulation）

用信息信号调制载波相位。是数字PSK的模拟前身，实际独立使用较少。

✅ 优点
  • 电路实现简单，成本低
  • FM抗幅度干扰能力强
  • 接收机设计成熟
  • 覆盖范围广（长波AM）
❌ 缺点
  • 噪声无法再生去除，逐级累积
  • 频谱效率低，带宽浪费大
  • 无法加密，安全性差
  • 不适合数据业务
AM / FM / PM 波形对比
🏛️ 历史地位：1946年首个移动无线电话系统（0G）使用点对点模拟通信；1979年日本NTT推出1G蜂窝网络，采用FM模拟语音。模拟调制奠定了无线通信的工程基础。
🔗 来源：Wikipedia - List of wireless network technologies

📶ASK — 幅移键控

幅移键控（Amplitude Shift Keying）是最基础的数字调制方式，通过改变载波信号的幅度来编码数字信息，其他参数（频率、相位）保持不变。最特殊的形式是开关键控（OOK，On-Off Keying），即有信号代表"1"，无信号代表"0"。

调制原理：s(t) = An · cos(2πfct)，其中 An 根据数字码元取不同幅度值。
2ASK（OOK）调制波形与星座图
✅ 优点
  • 电路实现极简单
  • OOK解调无需载波恢复
  • 低功耗，适合短距IoT
  • 光纤OOK可达极高速率
❌ 缺点
  • 幅度对噪声和衰落极敏感
  • 频谱效率低（1 bit/symbol）
  • 多径环境表现差
  • 需要AGC控制，实现复杂化

要解决的问题与适用场景

解决问题：在极低成本约束下实现基本数字通信，无需复杂同步电路。

适用场景：遥控器（红外/射频）、近场通信（NFC）、光纤通信（OOK调制是主流）、RFID低速标签、LoRa前身FSK的对比基准。

🔗 来源：CSDN - 网络调制技术对比表

🎵FSK — 频移键控

频移键控（Frequency Shift Keying）通过改变载波频率来编码数字信息，幅度保持恒定。二进制FSK（BFSK）用频率f₁代表"1"，f₂代表"0"。连续相位FSK（CPFSK）在频率切换时保持相位连续，避免产生宽带旁瓣干扰。

调制原理：s(t) = A·cos(2πfit + φ)，fi ∈ {f₁, f₂, ..., fM}，幅度A恒定，频率根据码元切换。
2FSK 调制波形示意
✅ 优点
  • 恒定包络，抗幅度噪声强
  • 抗衰落能力比ASK强
  • 可用非相干检测，电路简单
  • LoRa中CSS扩展FSK实现超远距离
❌ 缺点
  • 频谱效率低，需要更大带宽
  • 高阶MFSK带宽随M指数增长
  • 比PSK需要更高的Eb/N₀
  • 不适合高速宽带通信

适用场景

低速无线遥测（气象、工业传感）、LoRa/LoRaWAN物联网（CSS变体）、旧式Modem（Bell 103标准）、RFID系统、蓝牙基础率（BT-BR）、GSM中早期数据业务。

🔗 来源：Cadence System Analysis

🔄PSK — 相移键控

相移键控（Phase Shift Keying）通过改变载波相位来编码信息，幅度保持恒定。BPSK（二进制PSK）是最简单形式，QPSK每符号携带2 bits，8PSK每符号3 bits。DPSK（差分PSK）通过相对相位变化而非绝对相位来编码，避免需要相干载波参考。

BPSK / QPSK / 8PSK 星座图对比
EDGE（2.75G）核心：GSM网络引入8PSK调制，使数据速率从GPRS的114 kbps提升至384 kbps（理论），是2G向3G过渡的关键技术。标准由ETSI于2003年发布。
🔗 来源：Wikipedia - List of wireless network technologies
✅ 优点
  • 恒定包络，功放效率高
  • 抗噪声能力优于ASK/FSK
  • 频谱效率随阶数提升
  • BPSK/QPSK是卫星通信黄金标准
  • DPSK无需相干接收，简化实现
❌ 缺点
  • 高阶PSK（>8PSK）星座点密集，抗噪差
  • 需要精确载波相位同步
  • 8PSK以上频谱效率被QAM超越
  • DPSK会有约3dB性能损失

适用场景

BPSK/QPSK：卫星通信（DVB-S）、深空探测（NASA）、Wi-Fi控制帧（802.11b）；QPSK：3G UMTS下行链路；8PSK：EDGE（2.75G）；DPSK：蓝牙EDR模式（BT 2.0+）。

⬡QAM — 正交幅度调制

正交幅度调制（Quadrature Amplitude Modulation）同时调制载波的幅度和相位，将两路正交（I路和Q路）的幅度调制信号叠加传输。16-QAM每符号4 bits，64-QAM每符号6 bits，256-QAM每符号8 bits，1024-QAM每符号10 bits。QAM是现代高速无线通信的频谱效率核心。

5G NR最高支持256-QAM（下行），1024-QAM已在讨论中。根据Ericsson技术评论，1024-QAM可能成为未来NR规范的一部分，固定点对点回传已使用高于256-QAM的调制阶数。
🔗 来源：Ericsson Technology Review - 5G NR Physical Layer
16-QAM 与 64-QAM 星座图
✅ 优点
  • 频谱效率极高（log₂M bit/symbol）
  • 与OFDM结合是4G/5G核心
  • 自适应调制可随信道动态切换
  • 256-QAM用于5G毫米波密集场景
❌ 缺点
  • 非恒定包络，功放须线性工作
  • 高阶QAM要求极高SNR
  • 对相位噪声和IQ失衡敏感
  • 解调复杂度高，功耗大

〰️MSK / GMSK — 最小频移键控

最小频移键控（MSK，Minimum Shift Keying）是一种连续相位频移键控（CPFSK），调制指数精确为0.5——这是保持两个频率符号正交的最小频率间隔，由Collins Radio的Melvin Doelz和Earl Heald于1950年代末开发。

高斯最小频移键控（GMSK）在MSK基础上，将数字数据流通过高斯滤波器进行预调制滤波，平滑相位过渡，大幅压窄旁瓣。GSM标准（2G）采用GMSK调制。

维基百科（Wikipedia - Minimum-shift keying）："GMSK is used in the GSM mobile phone standard. In MSK the difference between the higher and lower frequency is identical to half the bit rate. This is the smallest FSK modulation index that can be chosen such that the waveforms for 0 and 1 are orthogonal."
🔗 来源：Wikipedia - Minimum-shift keying
MSK vs GMSK 相位轨迹对比
✅ 优点
  • 恒定包络，可用高效非线性功放
  • 相位连续，旁瓣极低
  • GMSK频谱紧凑，相邻频道干扰小
  • 适合电池供电移动终端（省电）
❌ 缺点
  • 频谱效率低，仅1 bit/symbol
  • 高斯滤波引入码间串扰（ISI）
  • 不适合高速数据业务
  • 已被更高效调制方案取代（LTE以后）

GSM为何选择GMSK？

根据权威分析，GSM选择GMSK因其恒定包络特性允许使用C类功率放大器，功率效率高，延长手机电池寿命。同时GMSK频谱紧凑满足ETSI对邻道干扰的严格规范。这是在1990年代初处理器能力有限条件下的最优工程选择。

🔗 来源：Explain QAM, MSK, and GMSK modulation

🛰️APSK — 幅度相移键控

APSK（Amplitude Phase Shift Keying）是幅度与相位的联合调制，信号点分布在多个同心圆环上，与QAM方格阵列相比，APSK的峰均功率比（PAPR）更低，对非线性功放（如卫星上的行波管放大器TWTA）的失真更鲁棒。

16-APSK 与 32-APSK 星座图（DVB-S2标准）
✅ 优点
  • PAPR低于同阶QAM
  • 对卫星功放非线性失真鲁棒
  • DVB-S2官方标准
  • 深空、军事通信场景优选
❌ 缺点
  • 接收端解调算法复杂
  • 频谱效率略低于等阶QAM
  • 不适合多径丰富的地面无线

适用场景：DVB-S2卫星广播（ETSI EN 302 307标准）、深空探测、军事卫星通信（MILSATCOM）。

🔗 来源：Cadence System Analysis - APSK Constellation

🌊OFDM — 正交频分复用

正交频分复用（Orthogonal Frequency Division Multiplexing）将高速数据流分割为多个并行低速子流，每路子流调制一个子载波，所有子载波在频率上相互正交。通过插入循环前缀（CP）抵抗多径衰落，通过快速傅里叶变换（FFT/IFFT）高效实现。OFDM是4G LTE和5G NR的核心波形技术。

3GPP/IEEE权威来源：3GPP在4G LTE Release 8（2008年）采用OFDMA（OFDM+多址接入）作为下行波形。5G NR采用CP-OFDM并配以可扩展参数集（子载波间距15/30/60/120/240 kHz）以适应从低频到毫米波的广泛频段。
🔗 来源：IEEE Communications Standards Magazine - 5G NR MIMO
OFDM 子载波正交性原理与循环前缀
✅ 优点
  • 强抗多径衰落（CP消除ISI）
  • 频谱效率高，子载波利用率高
  • FFT/IFFT数字实现高效
  • 灵活支持OFDMA多用户接入
  • 适配MIMO空间复用
❌ 缺点
  • 高峰均比（PAPR），功放效率低
  • 对载波频偏（CFO）和相位噪声敏感
  • 需要精确时频同步
  • CP引入约7-25%的频谱开销

📱SC-FDMA — 单载波频分多址

SC-FDMA（Single-Carrier Frequency Division Multiple Access）是专为移动终端上行链路设计的调制多址方案，本质是在OFDM基础上增加一个DFT预编码步骤（也称DFT扩展OFDM，DFT-s-OFDM）。其核心优势是将OFDM的高PAPR大幅降低，使终端功放工作在更高效率点，延长电池续航。

3GPP LTE上行标准：LTE（3GPP Release 8）规定上行采用SC-FDMA，下行采用OFDMA。5G NR上行同时支持CP-OFDM和DFT-s-OFDM（π/2-BPSK+DFT-s-OFDM用于mMTC低速率场景进一步降低PAPR）。
SC-FDMA 与 OFDMA 发射链路对比
✅ 优点
  • PAPR远低于OFDM（约低6dB）
  • 功放效率高，延长手机续航
  • 保留OFDM频率选择性调度优势
  • 适合上行功率受限场景
❌ 缺点
  • DFT预编码限制了灵活调度
  • 接收端均衡复杂度略高
  • 不支持OFDMA那样灵活的多用户复用
🔗 来源：Wikipedia - List of wireless network technologies（LTE章节）

👥OFDMA / MU-MIMO — 多用户接入

OFDMA（Orthogonal Frequency Division Multiple Access）是OFDM的多用户扩展，将子载波资源按时频资源块（RB）分配给不同用户。MU-MIMO（多用户MIMO）通过空间域隔离同时服务多用户。两者结合是LTE、5G NR和Wi-Fi 6/7的核心多用户技术。

OFDMA 资源网格 — 时频二维资源分配
Wi-Fi 6（802.11ax）：IEEE 802.11ax首次在Wi-Fi中引入OFDMA，将信道分割为资源单元（RU），多用户可同时在不同RU上传输，大幅提升密集场景容量。Wi-Fi 7（802.11be）进一步引入Multi-Link Operation（MLO）和4096-QAM。
✅ OFDMA 优点
  • 精细时频资源粒度调度
  • 支持频率选择性调度增益
  • 与MU-MIMO结合实现更高容量
  • Wi-Fi 6密集AP场景效率大幅提升
❌ OFDMA 缺点
  • 需要精确的时频同步
  • 调度算法复杂，信令开销大
  • 继承OFDM的高PAPR问题

Massive MIMO演进

根据IEEE/3GPP文献，MIMO-OFDM自3GPP Release 8（2008年LTE）起成为移动网络核心技术并延续至5G NR。5G NR Massive MIMO支持最多256根天线（64T64R常见部署），通过波束赋形实现空间多路复用，显著提升小区容量。

🔗 来源：Wikipedia - MIMO（3GPP Release记录）
🔗 来源：PMC - Technology Trends for Massive MIMO towards 6G

🔀NOMA — 非正交多址接入

非正交多址接入（NOMA，Non-Orthogonal Multiple Access）允许多个用户共享相同的时频资源，通过功率域（PD-NOMA）或码域（CD-NOMA）区分用户。接收端使用连续干扰消除（SIC，Successive Interference Cancellation）依次解码各用户信号。NOMA在3GPP Release 14中作为多用户叠加传输（MUST）被标准化研究。

PD-NOMA 功率域多址原理
✅ 优点
  • 无需额外频谱，提升连接密度
  • 小区边缘用户质量提升（高功率）
  • 与OFDM/OFDMA结合灵活
  • 满足mMTC大连接需求
❌ 缺点
  • SIC误差传播，影响整体性能
  • 需要精确的功率控制和信道估计
  • 用户数增加时SIC复杂度激增
  • 3GPP标准化进程缓慢（仍在研究阶段）
🔗 来源：TechRxiv - Multiple Access Framework for 6G（3GPP Release 14 NOMA研究）

🚗OTFS — 正交时频空间调制

正交时频空间（OTFS，Orthogonal Time Frequency Space）调制是为高速移动场景（V2X车联网、高铁通信、无人机）设计的新型调制技术，候选6G物理层波形之一。OTFS在延迟-多普勒（Delay-Doppler）二维域处理信号，而非OFDM的时频域，使高速移动下的信道响应呈现稀疏紧凑特性，实现全时频分集。

OTFS vs OFDM：延迟-多普勒域 vs 时频域
✅ 优点
  • 高速移动场景下性能远超OFDM
  • 信道在DD域稀疏，估计精度高
  • 利用全时频分集，提升链路质量
  • V2X、高铁、无人机理想候选
❌ 缺点
  • 接收机复杂度高（二维信号处理）
  • 与现有OFDM系统不兼容
  • 大规模部署仍面临实现挑战
  • 尚无商用标准（6G研究阶段）
6G研究前沿：根据arxiv/IEEE文献，OTFS被广泛认为是6G物理层候选技术，尤其适合高移动性场景。6G研究始于2010年代末，目标商用时间约2028年。
🔗 来源：arXiv 2301.00933 - OTFS-SCMA for 6G

⏳技术演进时间轴

无线通信调制技术演进路线图（1920→2030+）
1920s-1970s

模拟调制时代

AM广播（1920s），FM调频广播（1933年Armstrong发明）。模拟通信系统主导，电路简单但频谱效率低。

1979-1990（1G）

1G模拟蜂窝 — FM调频

日本NTT（1979）推出世界首个蜂窝网络，采用频率调制（FM）传输模拟语音。美国AMPS、欧洲NMT/TACS相继推出。峰值速率<10kbps。无加密，安全性差。

1991-2000（2G）

2G数字通信 — GMSK / 8PSK

GSM采用GMSK（BT=0.3高斯滤波），数字语音+SMS成为标准。GPRS（2000）引入数据业务，EDGE（2003）升级8PSK（2.75G），速率达384kbps。

2001-2009（3G）

3G宽带移动 — QPSK/QAM + W-CDMA

UMTS/W-CDMA采用QPSK调制结合码分多址。HSDPA（3.5G，2005）引入16-QAM，HSPA+（3.75G，2008）达64-QAM，峰值42Mbps。

2009-2018（4G）

4G LTE — OFDMA + MIMO + 64/256-QAM

LTE（2009）以OFDMA+SC-FDMA为核心，引入MIMO-OFDM（3GPP Release 8），64-QAM峰值150Mbps。LTE-A Pro引入256-QAM、载波聚合，峰值1Gbps。IEEE 802.11n（2009）将OFDM带入Wi-Fi。

2019-至今（5G）

5G NR — CP-OFDM + 256-QAM + Massive MIMO

5G NR（3GPP Release 15+）采用CP-OFDM，上下行均支持，可扩展参数集覆盖FR1/FR2。最高256-QAM，Massive MIMO最多256天线，峰值20Gbps。Wi-Fi 6（802.11ax）引入OFDMA多用户接入。

~2028（6G研究）

6G候选技术 — OTFS / NOMA / 1024-QAM+ / THz

目标速率>1Tbps，候选技术包括OTFS（延迟-多普勒域调制）、NOMA（非正交多址）、Sub-THz/THz通信（100GHz-3THz）、RIS（可重构智能超表面）、AI驱动自适应调制。

💡本质总结 — 第一性原理

第一性原理（First Principles）方法：剥离所有工程表象，回归最根本的物理约束与信息论极限，从底层逻辑推导调制技术演进的内在必然性。

🧱 第一原理一：香农信道容量定理（Shannon Limit）

C = B · log₂(1 + SNR)

信道容量C（bps）由带宽B（Hz）和信噪比SNR共同决定。这是不可逾越的物理上限。所有调制技术的演进本质上是在给定带宽和SNR约束下逼近香农极限的工程努力。

推论：频谱资源日益稀缺 → 必须最大化 bits/Hz → 推动高阶QAM（每符号携带更多比特）。但高阶QAM要求更高SNR（星座点更密），这就是为什么5G密集小区（近距离、高SNR）才能使用256-QAM。

🧱 第二原理：无线信道的本质是时变多径系统

电磁波在空间传播必然遭遇反射、衍射、散射，产生多径效应（频率选择性衰落）；发射/接收端移动产生多普勒效应（时间选择性衰落）。这两种效应是推动调制技术进化的核心物理约束：

• 对抗多径 → OFDM：将宽带信道分割为多个窄带平坦子信道，用CP消除ISI。这是从单载波到多载波的根本动力。

• 对抗高速移动多普勒 → OTFS：在延迟-多普勒域建模，信道在此域呈准静态稀疏，是比时频域更自然的表示。

🧱 第三原理：功率效率与频谱效率的根本性权衡

功率效率 ↑ ←→ 频谱效率 ↓（固定误码率下）

这是信息论中编码增益与调制阶数之间的基本权衡。恒定包络调制（FSK、GMSK、MSK）功率效率极高（允许非线性功放），但频谱效率低。高阶QAM频谱效率极高，但需线性功放（回退工作），功率效率低。

这一矛盾直接解释了：2G选GMSK（手机电池为重）、4G/5G用QAM（追求速率）、5G上行用SC-FDMA（手机省电）、卫星用APSK（非线性功放环境）。

🧱 第四原理：多用户本质是空间-时间-频率-码字四维资源的分割与共享

无线通信的根本任务是将有限的四维资源（频谱、时间、空间/天线、码字）分配给多个用户。调制多址技术演进的本质是维度扩展与利用：

• 频分（FDMA）→ 时分（TDMA）→ 码分（CDMA）→ 正交频分（OFDMA）→ 空间（MIMO）→ 非正交叠加（NOMA）→ 延迟-多普勒域（OTFS）

每一次维度扩展都带来容量的质的跃升，同时也带来对应的接收机复杂度增加。随着算力的指数增长（摩尔定律），复杂度约束持续放宽，技术维度不断扩展。

🧱 第五原理：调制技术演进受计算能力约束，不仅是物理约束

OFDM的FFT在1960年代就已理论存在，但受制于当时的数字信号处理芯片能力，直到2000年代才在移动通信中规模部署。Massive MIMO的波束赋形算法需要实时矩阵运算，直到5G时代的专用芯片才得以实现。OTFS的二维信号处理目前仍是6G研究的挑战，但随着AI芯片和专用硬件发展，必将走向实用。

可用调制复杂度上限 ∝ 可用计算能力（FLOPS）/ 符号周期

结论：无线调制技术的演进不是随机的，而是在香农极限、信道物理特性、功率约束和计算能力四重约束下，系统性地逼近理论最优解的必然过程。未来6G的关键不仅是发明新波形，更是通过AI赋能的自适应调制实现对香农极限的动态趋近。

第一性原理总结：四大约束驱动调制演进
⚡ 核心洞察：无线通信调制技术从来不是"发明"出来的，而是在香农信息论的理论上限指引下，针对信道物理特性、功率约束和计算能力三重现实边界，进行工程权衡的必然演化。每一代技术的跃升，本质上都是某一约束条件的放松（更好的算法、更强的芯片、更密的基站、更高的SNR）带来的新平衡点。
