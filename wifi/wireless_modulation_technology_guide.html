一、什么是调制与解调

在无线通信系统中，调制（Modulation）是将携带信息的基带信号"加载"到高频载波上的过程，使其适合在无线信道中传输；解调（Demodulation）则是在接收端从载波中恢复原始信息的逆过程。调制器和解调器合称为调制解调器（Modem）。

图1：调制解调基本通信系统模型

载波信号的一般数学表达式为：s(t) = A·cos(2πfct + φ)，其中包含三个可被调制的参量：

  • 幅度 A → 幅度调制（AM / ASK）
  • 频率 fc → 频率调制（FM / FSK）
  • 相位 φ → 相位调制（PM / PSK）

二、为什么需要调制

数字基带信号不能直接在无线信道中高效传输，调制的核心目的包括：

调制的三大核心目的
  1. 频率搬移，匹配天线：基带信号频率很低，需要搬移到高频载波上，使天线尺寸可以缩小到实用范围（天线长度 ≈ λ/4）。
  2. 多用户共享频谱：通过将不同用户的信号调制到不同载波频率上，实现频分复用（FDM），避免相互干扰。
  3. 承载更多信息：采用高阶调制技术（如 QAM），一个符号可以携带多个比特，在有限带宽内提高数据传输速率和频谱效率。
图2：调制将基带信号搬移到不同载波频率，实现频分复用

三、幅移键控 (ASK)

幅移键控（Amplitude Shift Keying, ASK）通过改变载波的幅度来表示数字数据。最简单的形式为OOK（On-Off Keying），即"1"时有载波输出，"0"时无载波。

调制原理

ASK的调制信号数学表达式：s(t) = A · m(t) · cos(2πfct)，其中 m(t) 为数字调制信号，fc 为载波频率。

图3：ASK（OOK）调制波形示意
ASK 特性总结

优点 电路简单、实现成本低、带宽效率高

缺点 抗噪声干扰能力差、功率效率低、易受信道衰落影响

适用场景 遥控器、RFID、光通信（OOK）、低速低成本系统

要解决的问题 如何用最简单的方式实现数字信号的无线传输

四、频移键控 (FSK)

频移键控（Frequency Shift Keying, FSK）利用不同频率的载波来表示不同的数字数据。数字信号为"1"时使用频率 f1，为"0"时使用频率 f2。

调制原理

FSK信号表达式：s(t) = A · cos(2π(fc + Δf · m(t))t)，其中 Δf 为频偏。

图4：2FSK调制波形示意——不同频率代表不同比特
FSK 特性总结

优点 抗噪声性能好、实现相对简单、恒包络（功放友好）

缺点 占用带宽较大、频谱利用率低

适用场景 LoRa、RFID、低速Modem、蓝牙BR、寻呼系统、遥测

要解决的问题 在高噪声环境下实现可靠的低速数据传输

五、相移键控 (PSK)

相移键控（Phase Shift Keying, PSK）通过改变载波信号的相位来传递数据。常见变种包括 BPSK（2相）、QPSK（4相）、8PSK 以及 DPSK（差分PSK）等。

BPSK 与 QPSK

BPSK：使用 0° 和 180° 两个相位，每个符号携带 1 bit。抗噪性最强，但频谱效率最低。

QPSK：使用 4 个相位（45°, 135°, 225°, 315°），每个符号携带 2 bit。数据速率是 BPSK 的两倍。

DPSK：利用前后码元之间的相位差（而非绝对相位）传递信息，无需载波同步，降低了接收机复杂度。

图5：BPSK调制波形——相位 0° 和 180° 分别代表比特 0 和 1
PSK 特性总结

优点 抗噪性强、功率效率高、频谱效率中等到高

缺点 需要精确的载波同步（DPSK除外）、高阶PSK对相位噪声敏感

适用场景 Wi-Fi（BPSK/QPSK）、卫星通信（QPSK）、LTE（QPSK）、蓝牙EDR（DPSK）

要解决的问题 在噪声和功率受限条件下实现可靠传输

六、正交幅度调制 (QAM)

正交幅度调制（Quadrature Amplitude Modulation, QAM）是 ASK 和 PSK 的联合调制，同时改变载波的幅度和相位来传输数据。两个正交载波（I路和Q路）各自做幅度调制后叠加，每个符号可以携带更多比特。

常见阶数

  • 16-QAM：4 bit/符号，16种振幅-相位组合
  • 64-QAM：6 bit/符号，用于 4G LTE、Wi-Fi
  • 256-QAM：8 bit/符号，用于 5G NR、Wi-Fi 6
  • 1024-QAM：10 bit/符号，用于 Wi-Fi 7、5G 演进

QAM阶数越高，频谱效率越高，但对信噪比（SNR）的要求也越高。星座点越密集，越容易被噪声干扰导致误判。

QAM 特性总结

优点 频谱效率极高（"频谱效率之王"）、数据速率高

缺点 需要高信噪比、对功放线性度要求高、实现复杂

适用场景 4G/5G/6G、Wi-Fi 6/7、有线电视、光纤通信

要解决的问题 在有限带宽内最大化数据传输速率

七、最小频移键控 (MSK / GMSK)

MSK（Minimum Shift Keying）是一种特殊的连续相位 FSK，频偏指数恰好为 0.5，使相邻符号之间相位连续变化，带外辐射极小。

GMSK（Gaussian MSK）在 MSK 之前加了一个高斯低通滤波器，进一步平滑相位转换，减小频谱旁瓣，是 GSM（2G）蜂窝系统的标准调制方式。

核心特征：恒定包络

GMSK 信号的幅度恒定不变，这意味着可以使用高效的非线性功率放大器（C类功放），极大提高功放效率、降低功耗——这对移动终端电池寿命至关重要。

MSK/GMSK 特性总结

优点 恒定包络、功放效率高、抗干扰强、带外辐射小

缺点 频谱效率较低（每符号仅 1 bit）

适用场景 GSM（2G）、蓝牙BR、军事通信

要解决的问题 在功率受限和频谱资源有限的场景下平衡效率与可靠性

八、正交频分复用 (OFDM)

正交频分复用（Orthogonal Frequency Division Multiplexing, OFDM）是多载波调制（MCM）的一种特例。其核心思想是将宽带信道分成若干正交的窄带子载波，将高速数据流分解为多路低速并行子数据流，分别在各子载波上传输。

为什么OFDM是4G/5G的基石？

  • 抗多径衰落：每个子载波带宽远小于信道相干带宽，近似平坦衰落，消除符号间干扰（ISI）
  • 高频谱效率：子载波正交重叠，无需保护频带
  • 高效实现：发射端使用 IFFT、接收端使用 FFT，硬件实现高效
  • 灵活资源分配：可对不同子载波使用不同调制阶数（如QPSK、16QAM、64QAM）

OFDM 特性总结

优点 强抗多径能力、高频谱效率、灵活资源分配、FFT高效实现

缺点 高峰均功率比（PAPR）、对频偏和相噪敏感、需精确同步

九、SC-FDMA / DFT-s-OFDM

SC-FDMA（Single Carrier Frequency Division Multiple Access）又称 DFT-s-OFDM（DFT-spread-OFDM），是一种单载波特性的多载波传输方案。它在 OFDM 调制之前先进行一次 DFT 预编码，从而使发射信号保持类似单载波的低峰均比特性。

与 OFDM 的关键区别

SC-FDMA 的 PAPR 显著低于 OFDM（典型改善 3dB 以上），这使得终端设备可以使用更高效的功率放大器，延长电池寿命。因此 4G LTE 和 5G NR 在上行链路均优先采用此方案。

SC-FDMA 特性总结

优点 低峰均比（省电）、兼具 OFDM 抗多径能力、频域灵活调度

缺点 频谱效率略低于 OFDM、接收端均衡复杂度较高

适用场景 4G LTE 上行、5G NR 上行（DFT-s-OFDM 可选）

要解决的问题 在保持 OFDM 优势的同时降低终端功耗

十二、星座图图解

星座图（Constellation Diagram）是表示数字调制方式的直观工具，横轴为同相分量（I），纵轴为正交分量（Q）。每个点代表一个符号，点的位置由幅度和相位决定。星座点越密集，频谱效率越高，但对噪声越敏感。

十三、新型波形技术（5G/6G 前沿）

在 3GPP 标准化过程中，除 CP-OFDM 外，业界还提出了多种改进波形方案，主要通过在 OFDM 基础上增加额外滤波来获得更好的频谱约束性。以下是主要候选技术：

FBMC（Filter Bank Multi-Carrier，滤波器组多载波）

对每个子载波单独滤波，彻底消除子载波间干扰和带外泄漏，无需循环前缀（CP），频谱效率进一步提升。但滤波器设计复杂，与 MIMO 技术适配困难。

UFMC（Universal Filtered Multi-Carrier，通用滤波多载波）

对子载波组（子带）进行滤波而非逐个子载波滤波，在复杂度和频谱约束之间取得折中。适合 5G URLLC（超低延迟高可靠通信）和频谱共享场景。

GFDM（Generalized Frequency Division Multiplexing，广义频分复用）

非正交的多载波方案，可使用灵活的脉冲成形滤波器，支持更短的 TTI，适用于低延迟场景，但接收端干扰消除较复杂。

5G NR 最终选择

经过综合权衡，3GPP Release 15 最终确定：上下行统一采用 CP-OFDM 多载波波形，上行对功率受限场景提供 DFT-s-OFDM 作为可选方案。新型波形技术（FBMC、UFMC等）虽有优势，但增加的实现复杂度和与现有 MIMO 技术的兼容性问题使其未被 5G R15 标准采纳，可能在 6G 中发挥更大作用。

十四、第一性原理总结

回到最本质的物理规律，所有调制解调技术的演进都可以从以下几个第一性原理推导出来：

图11：调制技术演进的第一性原理框架
原理一：香农信道容量定理 — 一切的上限

C = B · log₂(1 + SNR)

信道容量 C（比特/秒）由带宽 B 和信噪比 SNR 共同决定。这告诉我们：

  • 提高频谱效率（每赫兹传更多比特）必须以更高信噪比为代价 → 这解释了从 BPSK 到 1024-QAM 的演进逻辑
  • 在 SNR 有限时，低阶调制（BPSK/QPSK）是最优选择 → 这解释了为什么卫星和边缘覆盖仍然使用低阶调制
  • 增大带宽也能提升容量 → 这解释了 5G 使用毫米波大带宽的动机
原理二：电磁波传播特性 — 为什么必须调制

基带信号频率太低，天线需要与信号波长相当才能高效辐射。对于 1 kHz 的语音信号，所需天线长度约 75 km，完全不可行。将信号调制到 GHz 载波上后，天线仅需 ~7.5 cm。调制的存在是由电磁波的物理传播特性所决定的。

原理三：能量守恒 — 效率与可靠性的永恒博弈

高阶调制（如 256-QAM）需要更高的信号发射功率来维持星座点间的足够距离，否则误码率急剧上升。恒包络调制（如 GMSK）允许使用非线性高效功放，功率效率高但频谱效率低。不存在同时最优化功率效率和频谱效率的调制方案——这是能量守恒的必然结果。

原理四：傅里叶变换 — OFDM 的数学之美

时域的乘法对应频域的卷积，时域的短脉冲对应频域的宽频谱。OFDM 正是利用 FFT/IFFT 这一数学工具，将频率选择性衰落的宽带信道分解为多个平坦衰落的窄带子信道，将一个"困难"的均衡问题转化为多个"简单"的均衡问题。这不是工程上的巧妙设计，而是时频对偶性这一数学原理的直接应用。

🔑 终极洞察

调制技术的整个演进史，本质上是人类在香农极限这个不可突破的物理天花板下，通过不断创新来逼近理论上限的过程：

  1. ASK/FSK/PSK → 解决了"能不能传"的问题（数字化）
  2. QAM → 解决了"如何传更多"的问题（高频谱效率）
  3. OFDM → 解决了"在恶劣信道中如何传"的问题（抗多径衰落）
  4. 自适应调制 → 解决了"如何动态最优传"的问题（根据信道条件实时切换调制阶数）
  5. 新型波形（FBMC等） → 试图解决"如何更灵活地传"的问题（频谱共享、异步接入）

每一代技术都不是凭空发明，而是对前一代技术在物理极限约束下发现的瓶颈的突破。理解了这些第一性原理，就能理解为什么调制技术必然走到今天这一步，也能预见未来的发展方向——在信噪比和带宽之间寻找新的平衡点。
